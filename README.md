# 简易计算器 🧮

一个使用纯 HTML、CSS 和 JavaScript 构建的现代化简易计算器，支持基本数学运算和键盘输入。


## ✨ 功能特点

-  **基本四则运算**：加、减、乘、除
-  **数字输入**：支持整数和小数运算
-  **正负号切换**：一键切换数字正负
-  **百分比计算**：快速百分比转换
-  **键盘支持**：支持键盘数字和运算符输入
- **美观界面**：现代化 UI 设计


## 🛠 技术栈

- **前端**：HTML5、CSS3、JavaScript (ES6+)
- **架构**：面向对象编程 (OOP)
- **样式**：CSS Grid 布局、CSS 动画
- **工具**：Git 版本控制

## 🎮 使用方法

### 鼠标操作
1. 点击数字按钮输入数字
2. 点击运算符按钮选择运算
3. 点击等号查看结果
4. 使用 AC 清除所有，✕ 退格

### 键盘操作
- **数字键** `0-9`：输入数字
- **运算符** `+`, `-`, `*`, `/`：基本运算
- **回车键** `Enter` 或 `=`：计算结果
- **退格键** `Backspace`：删除一位
- **ESC键** `Escape`：清除所有
- **小数点** `.`：输入小数
